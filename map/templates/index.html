Find my house





Features

Intuitive. Simple. Fast.

Find My House allows first home buyers within the UK to quickly view available property for purchase. Simply enter an address, and a map containing the sold property will be displayed. Click on a property marker, and the information is shown to you. There is no step 3 - that is it!

Clicking on a property marker reveals a basic picture of the property, as well as basic information such as address, price and property type. A link is provided to access the real estate agent for further contact.



Our software, your needs.

Find My House offers advanced filters so that only property that meets your criteria will be displayed. To access this feature, simply click the side menu button on the top right of the interface. The resulting drop-down menu will contain advanced filtering options to supplement the search for the right home.

The keyword filter allows for first home buyers to locate houses with specific properties of their need. This includes apartments, studios, access to lifts, access to gyms, and much more.

The Price interval filter allows for first home buyers to enter a price range for the property that they would like to view. This allows for a minimum price, as well as a maximum price.

The Distance filter allows for first home buyers to filter property close to a point of reference. This is extremely useful if buyers would like to only view property that is, for example, 30 minutes away from their workplace. First home buyers simply need to enter an address for the point of reference and a time limit to the maximum time taken to travel to the point. Buyers can then select whether they want the travel time to be accounted by driving, public transit or walking. The result is the display of houses that fit this criteria.
